Text Detection and Recognition in Video
2012
There are many works in the literature recently in text detection and recognition. There, however, are not many in the domain of video. In video, we seek to improve the performance by using temporal consistency and tracking.
BugOCR
2011
Links: blog demo

UC Berkeley Entomology department is looking to digitalize 1.2 millions specimen photos. The main task is to extract and organize texts in these photos. We are currently investigating methods of creating an interface that aid citizen scientists in extracting these data.
